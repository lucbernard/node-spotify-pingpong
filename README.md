node-spotify-pingpong
=====================

- Download atom-shell at https://github.com/atom/atom-shell/releases
- From the node-spotify-pingpong directory, run:

On Windows:

```cmd
$ .\atom-shell\atom.exe .\
```

On Linux:

```bash
$ ./atom-shell/atom ./
```

On Mac OS X:

```bash
$ ./Atom.app/Contents/MacOS/Atom ./
```
